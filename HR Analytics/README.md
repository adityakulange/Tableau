# HR Analytics Dashboard (Interactive Dashboard Using Tableau)

## Project Overview

This project focuses on HR Analytics using Tableau, aimed at understanding employee attrition patterns and key workforce metrics. The dashboard helps HR teams and management make data-driven decisions by visualizing employee trends, risk factors, and performance indicators.

## Dataset Used
- <a href="https://github.com/adityakulange/Tableau/blob/main/HR%20Analytics/HR%20Data.xlsx%20-%20HR%20data.csv">HR-Analytics-Dataset</a>

## Dashboard Features

1. KPI Cards
- Total Employees
- Attrition Count
- Attrition Rate
2. Visual Analysis
- Attrition by Age Group
- Attrition by Department
- Attrition by Salary Slab
- Experience vs Attrition
3. Interactive Elements
- Filters by Department, Job Role, and Gender
- Dynamic insights through visual storytelling
